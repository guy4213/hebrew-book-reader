# אתר הקראת ספרים בעברית

אתר בעברית (RTL) שמזהה ספר מתמונה או משם, מקריא אותו בקול ElevenLabs, מדגיש את המילה המוקראת בזמן אמת, ומציג את הטקסט גם כתצוגת ספר וירטואלי הפיך-דפים וגם כתצוגת קריאה רגילה.

## זרימת משתמש

1. **הוספת ספר** — שתי דרכים:
   - העלאת תמונה של כריכת הספר (מהמצלמה או מהמחשב) → זיהוי שם הספר ע"י Gemini Vision
   - הקלדת שם הספר ידנית
2. **התאמת תוכן** — אם הספר קיים בספרייה הדיגיטלית של האתר → טוען את הטקסט המלא. אחרת → המשתמש מעלה PDF/טקסט (או OCR אוטומטי מצילומי עמודים).
3. **קריאה והקראה** — בחירת תצוגה (ספר וירטואלי / קריאה רגילה), לחיצה על נגן, והאזנה תוך כדי הדגשת המילה הנקראת.

## מסכים עיקריים

- **בית** — ספרייה אישית של ספרים שנוספו + כפתור "הוסף ספר" + חיפוש
- **מסך הוספה** — העלאת תמונה / הקלדת שם / העלאת קובץ תוכן
- **מסך הקורא** — תצוגת הטקסט + נגן + הגדרות + טוגל מצב תצוגה (ספר/רגיל)

## נגן הקראה

- ▶️ נגן/השהה, קפיצה ±15 שניות, פס התקדמות
- 🎚️ מהירות (0.5×–2×)
- 🔊 ווליום
- 🎙️ בחירת קול (קולות עבריים של ElevenLabs: Sarah, George, Charlie וכו')
- ⏱️ הדגשת המילה הנקראת בזמן אמת (highlighting) עם גלילה אוטומטית
- ⏭️ ניווט לפי פסקה/עמוד

## תצוגת ספר וירטואלי

- שני עמודים זה ליד זה (כמו ספר פתוח), אנימציית הפיכת דף
- מצב חלופי: קריאה רגילה (גלילה אנכית, טיפוגרפיה נעימה)
- שליטה בגודל פונט, צבע רקע (לבן/קרם/כהה)

## ארכיטקטורה טכנית

### Frontend (React + Tailwind, RTL)
- `src/pages/Library.tsx` — דף בית עם רשימת ספרים
- `src/pages/AddBook.tsx` — הוספת ספר חדש (העלאה / שם)
- `src/pages/Reader.tsx` — מסך קריאה והקראה
- `src/components/reader/BookView.tsx` — תצוגת ספר וירטואלי (פלאגין `react-pageflip`)
- `src/components/reader/PlainView.tsx` — תצוגת קריאה רגילה
- `src/components/reader/AudioPlayer.tsx` — בקרי נגן
- `src/components/reader/WordHighlighter.tsx` — הדגשת מילים

### Backend (Lovable Cloud + Edge Functions)
- `identify-book` — מקבל תמונה, קורא ל-Gemini Vision (Lovable AI Gateway) → מחזיר שם מחבר/כותר
- `tts-generate` — מקבל טקסט+voice+speed, קורא ל-ElevenLabs API → מחזיר MP3 + תזמוני מילים (`with-timestamps` endpoint) לסנכרון הדגשה
- `ocr-pages` — אופציונלי, מצילומי עמודים → טקסט (Gemini Vision)

### מסד נתונים
- `books` — ספרים של המשתמש (כותר, מחבר, cover_url, content_url)
- `reading_progress` — מיקום אחרון לכל ספר
- `storage buckets`: `book-covers`, `book-content`, `audio-cache`
- RLS: כל משתמש רואה רק את הספרים שלו

### הקראה עם הדגשת מילים
שימוש ב-`/v1/text-to-speech/{voiceId}/with-timestamps` של ElevenLabs — מחזיר MP3 + מערך תווים עם זמני התחלה/סיום. נחבר חותמות זמן למילים שלמות ונסנכרן עם `audio.currentTime`.

### קישי הקראה
חיתוך הטקסט לפסקאות (≈500–1500 תווים) והפעלת `request stitching` של ElevenLabs לחיבור חלק בין מקטעים. שמירת אודיו ב-Storage כדי לא לחייב פעמיים.

## דרישות חיצוניות מהמשתמש
- חיבור **ElevenLabs** דרך Lovable Cloud Connectors (נבקש בזמן הבנייה)
- הפעלת **Lovable Cloud** (לבסיס נתונים, אחסון ופונקציות)
